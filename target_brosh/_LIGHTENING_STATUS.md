## Мета
Брошура була «навантажена». Полегшуємо: **розмірні ряди → таблиці** (тип-банда: фото+опис типу → рядки «розмір × ціна»); **різні товари → картки**. НЕ все таблицями. Жовтий хірургічно, шрифти читабельні.

## ВАЖЛИВО — технічне
- `_build_catalog.py` + `_assets_*.txt` — **СТАРІ (червоне на білому), НЕ запускати** (затруть тему). Правимо HTML напряму.
- Зміни scope-нуті класом `.lite` на `<section class="page lite">`.
- Дані товарів парсяться з існуючих OLD-секцій у `/tmp/items.json` (парсер нижче; /tmp ефемерний — перезапустити в новій сесії).
- Прев'ю-рендер: вирізати потрібні секції + `<head>` у тимчасовий html → headless Chrome `--print-to-pdf` → `pdftoppm -png`. (Chrome-devtools MCP має зайнятий профіль — НЕ використовувати, лише CLI Chrome з тимчасовим --user-data-dir.)
- Скрін-рендер падає у фон; чекати появи PDF циклом, тоді конвертувати в PNG.
- Система версій: `versions/versions.js` (ліва панель). v1 = стан ДО полегшення.

## ФІНАЛЬНА СПЕЦИФІКАЦІЯ КАРТКИ (різні товари)
Секція: `<section class="page lite">` → `.pad` → `runhdr`/`kicker`+`sec-title` → `.catbar` (назва + «N позицій») → `<div class="pgrid">` (2 колонки) з `.pcard`.
`.pcard` (фон surface-1, radius r-block, overflow hidden):
- `.ph.pcard-img` (min-height 100px) + опц. `.pcard-tag` («Хіт» для top, «Рек» для rec, жовта).
- `.pcard-body`: `<h4>` назва **16px** (БЕЗ дубля розміру в назві — обрізати хвіст «390x310» якщо лишається ≥2 слова); `.pc-meta` габарит body-шрифт 11px (формат «390×310 мм», БЕЗ товщини — вона в марках); `.pc-desc` 12px, перші 1–2 цілі речення, clamp 2 рядки; `.pc-price`.
- `.pc-price` (border-top hairline) → `.pc-strip` (`display:flex`, колонки `.pc-col{flex:1}` рівні на всю ширину, розділювач `.pc-col+.pc-col{border-left}`): кожна `.pc-col` (`.prem` якщо AR550, `.solo` якщо одна ціна) = `.pc-g` (BF_SUB_HEADLINE 11.5px, марка) + `.pc-v` (18px ціна) + `.pc-u` («грн» моно 9.5px).
- БЕЗ бірки «Ціна за одиницю» (прибрано — «грн» біля чисел достатньо).
- Артикул (sku) НЕ показуємо в картці (рекомендація; за потреби — дрібний «арт.»).

## ФІНАЛЬНА СПЕЦИФІКАЦІЯ ТАБЛИЦІ / ТИП-БАНДИ (розмірні ряди)
- `.ptbl` (або `.ptbl.compact` для довгих) → `.ptbl-cur` «Ціна за маркою сталі, **грн/од**» → `.ptr.head` (`.pt-h.size` «Розмір» + `.pt-h`×2 + `.pt-h.prem` AR550) → рядки `.ptr` (`.pt-size` + `.pt-val`×3, `.pt-val.dash` для «—», `.pt-tag` «Хіт»).

## ПАРСЕР даних (відтворити /tmp/items.json у новій сесії)
```python
import re,json
html=open('amtarget_catalog_prototype.html',encoding='utf-8').read()
body=html.split('</head>',1)[1]
token=re.compile(r'<h2 class="sec-title">(?P<cat>.*?)</h2>|<div class="catbar">(?P<sub>.*?)<span>(?P<cnt>.*?)</span>|<div class="hero"[^>]*><div class="ph hero-img"><span class="hero-badge (?P<badge>\w+)">|<div class="prow brk"[^>]*>')
cat=sub=cnt=None; pos=[]
for m in token.finditer(body):
    if m.group('cat') is not None: cat=re.sub('<.*?>',' ',m.group('cat')).strip()
    elif m.group('sub') is not None: sub=m.group('sub').strip(); cnt=m.group('cnt').strip()
    else: pos.append((m.end(),'hero' if m.group('badge') is not None else 'prow',cat,sub,cnt,m.group('badge') or ''))
def grab(s):
    e=body.find('price-req',s); return body[s:e if e>0 else s+1600]
def steels(ch): return [[a,int(b)] for a,b in re.findall(r'<span class="chip[^"]*">([^<]+)</span><span class="ps-val">(\d+)',ch)]
items=[]
for p,k,c,s,n,bg in pos:
    ch=grab(p)
    g=lambda rx: (re.search(rx,ch).group(1).strip() if re.search(rx,ch) else '')
    items.append(dict(cat=c,sub=s,cnt=n,kind=k,badge=bg,name=g(r'<h[34]>(.*?)</h[34]>'),sku=g(r'<span class="sku">(.*?)</span>'),dim=g(r'<span class="dim">(.*?)</span>'),desc=g(r'<p class="(?:pdesc|hero-desc)">(.*?)</p>'),prices=steels(ch)))
json.dump(items,open('/tmp/items.json','w'),ensure_ascii=False)
```
