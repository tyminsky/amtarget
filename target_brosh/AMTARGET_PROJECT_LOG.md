## 1. Клієнт і контекст

**AMTarget (A.M.TargeT)** — український виробник сталевих мішеней для стрільби.
- Локація: Требухів, Київська область. Сімейне виробництво **повного циклу**.
- Технологія: лазерне різання, точність **±0.1 мм**.
- Цифра: **10 000+** сталевих мішеней виготовлено.
- Сталь: **AR500 5 мм**, **AR500 10 мм**, **AR550 10 мм** (HBW 470–510 / 530–560). У фіді також трапляється AR500 20 мм (g150-s20).
- Сайти: `amtarget.net` (спорт/цивільний) + `amtarget.net/landing` (військовий).
- Військовий напрям (з лендинга): **63 військові частини**, працюють **з 2019**, відповідь **30 хв**, **Prozorro**, безготівка.
  - Підрозділи-замовники: 120-та ОБр ТрО, 427-ма ОБр БпС «Рарог», 67-ма ОМБр, 711-та бригада розмінування, в/ч 3059 НГУ, А2772, А3204, А3719, Т0120.
  - Відгуки: «Сталь тримає, геометрія рівна. Документи зробили швидко» — офіцер в/ч; «Якість на рівні, відмінно тримають інтенсивну стрільбу» — підполковник.
  - **Телефон: +38 (073) 50 50 855**, Telegram/WhatsApp, Пн–Сб 9:00–18:00.

### Подача товарів
- Категорії — **точно з фіду** (поле «Категорія»), не вигадані.
- Кожен товар = рядок: фото 88px + назва + артикул + розмір + **ціни окремо по кожній марці сталі** (шапка «Тип сталі / Ціна») + короткий опис із фіду.
- На початку кожної категорії — **2 топ-товари великими картками** (за міткою `top` у фіді; де немає — перші 2 як «РЕКОМЕНДУЄМО»).
- **Перемикач цін** угорі: «Показати ціни» ↔ «Ціна за запитом» (ховає ціни на товари дорожче 5000 грн).

## 3. Як влаштовано (для перезбірки)

Каталог **генерується скриптом** з фіду + ассетів. Не редагувати HTML вручну — правити джерела й перезапускати.

| Файл | Призначення |
|------|-------------|
| `feed - AMTarget.xlsx` | **Джерело даних** (аркуші `Gong - UA`, `Mishen - UA`): товари, ціни, розміри, сталь, категорії, описи, мітки `top` |
| `_build_catalog.py` | **Генератор**: читає фід + ассети, збирає HTML, нумерує сторінки, пагінує |
| `_assets_css.txt` | Усі стилі |
| `_assets_static.txt` | Статичні сторінки (обкладинка, про компанію, матеріали, сумісність, військові, безпека, контакти, controls, script). Блоки розділені маркером `@@@`, кожен має КЛЮЧ |
| `_assets_pagetmpl.txt` | Шаблон продуктової сторінки |
| **`amtarget_catalog_prototype.html`** | **Результат** (перезаписується щоразу) |

**Перезбірка:** `python _build_catalog.py` (потрібен `openpyxl`: `python -m pip install openpyxl`).

Логіка генератора: групує товари за категорією → у кожній обирає 2 «top» (або перші) як hero-картки, решту як рядки → будує потік (заголовок/підкатегорія/hero/товар) → пагінує по висоті (бюджет ~880px) з підписом «продовження» → підставляє наскрізні номери сторінок.

## 4. Хронологія рішень (цей чат)

1. Визначили проєкт, оцінили готовність до прототипу.
2. Зібрали **перший прототип** (14 блоків, картки + «флагман», перемикач цін).
3. Рішення клієнта: **тільки реальні категорії з фіду**, формат **список-рядки** (не «флагман»), ~10/категорію.
4. Перебудували генератором прямо з фіду.
5. Ціни: обрано **розбивку по кожній марці сталі** (одну категорію показали в обох варіантах для порівняння).
6. Збільшили фото до **88px**, додали шапку **«Тип сталі / Ціна»**, увели **пагінацію**.
7. Додали **короткі описи** з фіду в кожен товар.
8. **Рев'ю 8 персон-клієнтів** у 3 етапи з верифікацією → `persona_review_findings.md`.
9. Додали сторінку **«Працюємо з військовими»** на даних лендинга + реальний телефон.
10. Блок матеріалів: фото в картки → **дубль-сторінка** зі зразками в таблиці → опис у таблицю, картки прибрано.
11. Додали **тексти призначення сталі** + фото в кожен блок.
12. Додали блок переваг **«Чому сталь, а не папір»**.
13. Додали **2 топ-картки** на категорію (мітки `top` з фіду).
14. Нова **обкладинка «На полігоні»** (екшн-концепт, таглайн «Перевірені на передовій»).

## 5. Рев'ю 8 персон-клієнтів (підсумок)

Повний звіт: **`persona_review_findings.md`**. Метод: 3 етапи (дизайн персон → аналіз → рекомендації) з верифікацією у фіді.

**Персони:** IPSC-спортсмен · початківець · власник тиру · офіцер ЗСУ · тендерник ТрО · волонтер/фонд · інструктор · дилер.

**Консенсус must-have (вимагали майже всі):**
1. Реальні фото (зараз заглушки) — 8/8
2. Живі контакти — 8/8 *(телефон уже додано)*
3. Терміни виготовлення + доставка — 8/8
4. Гарантія в цифрах + ресурс у пострілах — 6/8
5. Фікс «брудних» даних — 6/8
6. Готові комплекти «під задачу» — 6/8
7. Вага + точні габарити в рядку

**Нові ідеї персон:** QR→відео+ціна онлайн · вартість пострілу (TCO) · сервіс заміни пластин · брендування/co-brand · демо-майданчик · позиція «партнер на сезон» · каталог за логікою вправ · шаблон звіту донорам · тендерний тех-додаток · MAP-політика дилера.
