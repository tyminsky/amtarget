# Візуальний аналіз каталогів конкурентів
### Для проекту каталогу AMTarget · 12 червня 2026

> **Метод:** кожен документ відкрито в перегляді PDF, проаналізовано візуально (обкладинка, зміст, типова верстка, подача продуктів, типографіка, колір). Нижче — що саме побачено на сторінках, а не переказ тексту.

> **Покриття:** глибоко розібрано 8 каталогів/брошур, що охоплюють усі дизайн-архетипи ринку: Eemann Tech, QT Targets, Birchwood Casey, Meggitt, Action Target (LE/Military, Commercial, Store, AutoTargets). Решта 3 матеріали Action Target (Range Customization, Range Services, ARCAS) — це брошури з тією самою дизайн-системою про інфраструктуру тирів, не релевантні до продуктів AMTarget.

## 1. Eemann Tech — Catalogue 2024 (204 стор.)
**Архетип: преміальний мінімалістичний каталог, організований за брендом зброї**

**Обкладинка:** темний фон, червоні геометричні «лазерні» лінії, велика шестерня з логотипом ET, вертикальний напис «CATALOGUE 2024». Технологічно, дорого, стримано.

**Структура (зі сторінки змісту):** About Us → Springs → Parts → Spare Screws → **Targets & Patches (стор. 175–185)** → Tools & Cleaning → Apparels. Тобто це насамперед каталог запчастин до зброї, а мішені — окремий розділ ~10 сторінок.

**Верстка продуктів:**
- **Флагман = одна сторінка = один продукт.** Сталева мішень Hardox займає цілу сторінку: фото на білому праворуч угорі, ліворуч — `Technical Specifications` маркованим списком (матеріал, товщина, діаметр, висота, вага), блок `Set Contains`, і **`Target Has Been Tested By`** з дистанціями й калібрами (Pistol 7 m, Rifle 100 m).
- **Варіанти = рядки.** Картонні мішені (IPSC / Mini / Micro / IDPA) показані стосом рядків: маленька міні-таблиця `Reference nr / Color / Quantity` + **кругла вирізка-callout** із зонами ураження.
- Колонтитули чергуються: вгорі «2024 + НАЗВА РОЗДІЛУ» (рудо-червоний), внизу «НАЗВА РОЗДІЛУ + EEMANN TECH».

**Що взяти для AMTarget:** принцип «один флагман — одна сторінка»; блок «протестовано на дистанціях/калібрах»; кругла вирізка для деталі зони ураження; стримана біло-чорна палітра з одним акцентним кольором.

## 2. QT Targets (Qualification Targets) — Catalog 2025 (29 стор.)
**Архетип: щільний прайс-лист, організований за стандартом/призначенням**

**Обкладинка:** фотографія паперової мішені-яблука зі стрілецькими окулярами й гільзами, логотип QT, «2025», «Targets.net» + телефон/email. Утилітарно, прямо.

**Верстка продуктів:**
- Кожен розворот = категорія з **червоним банером-заголовком** угорі (`IALEFI TARGETS`, `CRITICAL RESPONSE`) + номер сторінки великим.
- Щільна **сітка з десятків мініатюр** мішеней. Під кожною: код (жирним), опис, розмір (23"×35"), і **багаторівнева ціна за обсягом** (100 / 250 / 500 / 1 000 шт).
- Серія «Critical Response» — повноколірні фотореалістичні мішені (людина зі зброєю) для тактичної реалістичності.
- Колонтитул: «Page X · Toll Free · targets.net».

**Що взяти для AMTarget:** **відкрита таблиця цін за обсягом** — сильний сигнал для B2B/військових гуртових замовлень; організація за стандартом дисципліни; готовність показувати багато SKU охайно.

## 3. Birchwood Casey — Catalog 2024 (34 стор.)
**Архетип: роздрібно-дилерський каталог, фото-сітка для замовлення**

**Обкладинка:** світлий фон у напівтоновий «камуфляж» із крапок, червоний бейдж-логотип із прицілом, вертикальний напис «BIRCHWOOD CASEY», велике «2024». Сучасно, споживчо, бренд-орієнтовано.

**Верстка продуктів:**
- Шапка-«вкладка» темної кутастої форми: ліворуч «BIRCHWOOD CASEY» (червоний), праворуч назва розділу (`SHOOTING RESTS & BAGS`, `METAL TARGETS`, `METAL TARGET STANDS`).
- **Сітка 3 колонки** на білому. Кожен продукт: фото, НАЗВА (капс), **`ITEM #` (червоний) + `UPC`** код, короткий опис/спека. Цін немає (дилерські окремо).
- **Червоні «прапорці»-callout** на фото: `4 PACK`, `SELF-TIGHTENING GRIP`, `NESTS AND LOCKS TOGETHER`.

**Що взяти для AMTarget:** охайна сітка 3 колонки для багатьох дрібних SKU (гонги, тарілки); артикул для замовлення на видноті; кольорові «прапорці» для переваг/акцій; послідовні «вкладки» розділів для навігації.

## 4. Meggitt Defense — AGTS-36 Data Sheet (2 стор.)
**Архетип: корпоративний оборонний one-pager (data sheet)**

Це не каталог, а **дата-шит одного продукту** (буксирувана повітряна мішень для гарматної підготовки винищувачів). Цінний саме як шаблон військового one-pager.

**Верстка:**
- Слоган «Enabling the Extraordinary» вгорі-ліворуч, логотип MEGGITT праворуч.
- Мітка `DATA SHEET` (жовто-зелений акцент), велика назва продукту, фото праворуч.
- **Пронумеровані секції:** `01 Description · 02 Key features and benefits · 03 Applications · 04 Contact · 05 Specifications · 06 Outline details`.
- Соц-докази в тексті: «standard gunnery target for the U.S. Air Force», «More than 5,000 targets delivered».
- Сильний акцент на **відповідності військовим стандартам** (MIL-STD-810, MIL-STD-461, MIL-STD-882).
- Внизу примітка «specifications subject to change».

**Що взяти для AMTarget:** **готовий шаблон військового one-pager** для ЗСУ/Prozorro — Опис → Ключові переваги → Застосування → Специфікації → Контакт; цифри-докази; згадка стандартів (IPSC-відповідність, тип сталі/HBW як «стандарт»).

## 5. Action Target — Law Enforcement & Military Range Catalog (136 стор.)
**Архетип: преміальний каталог-рішення (еталон сегменту)**

**Обкладинка:** кінематографічне ч/б фото критого тиру з рядами силуетних мішеней у перспективі, синя вертикальна акцент-смуга, логотип + «LAW ENFORCEMENT AND MILITARY RANGE CATALOG». Стор. 2 — боєць у спорядженні з великим водяним знаком-логотипом.

**Структура (зі змісту) — за СИСТЕМАМИ/РІШЕННЯМИ, не за типом товару:**
Trusted Partner Warranty → Control Systems → Turning Targets → Running Man Systems → Reactive Targets (вкл. **Steel KD, Portable Steel Targets**) → Line of Fire → Target Retrievers → Bullet Traps → Safety Baffles → Shooting Stalls → Ventilation → Range Lighting → Shoot Houses → Modular Ranges → Range Security → Range Training → Range Services.

**Ключовий хід:** каталог **відкривається 3-річною гарантією** (`Trusted Partner Warranty — 3-Year Comprehensive Coverage`), а не переліком товарів. Спочатку довіра, потім продукт.

**Типова сторінка продукту (напр. «BOBBERS»):**
- Велика жирна назва + **вигодо-орієнтований підзаголовок** («GET FASTER WITH YOUR PISTOL»).
- Абзац-опис системи, далі **блоки фіч** у дві колонки (Pop-Up Action, Ballistic / Independent, Connected).
- **Порівняння варіантів поруч** (Bobber vs Bobber Xtreme).
- `LEARN MORE` + URL, рядок `FEATURES`.
- Повноекранне середовищне фото продукту в дії, **вертикальні червоні «вкладки» розділів** на краю сторінки.

**Що взяти для AMTarget:** організація за рішеннями + відкриття довірою (гарантія/«10 000 мішеней»); вигодо-орієнтовані заголовки замість сухих назв; великі фото «продукт у застосуванні»; вкладки розділів на краю для навігації.

## 6. Action Target — Commercial Shooting Range Catalog (96 стор.)
**Архетип: той самий бренд, інша аудиторія — інший тон**

**Обкладинка:** **кольорове, світле, привітне** фото сучасного комерційного тиру із синьо-підсвіченими системами подачі Genesis і брендованими паперовими мішенями. «COMMERCIAL SHOOTING RANGE CATALOG».

**Найважливіший висновок усього аналізу:** порівняйте з LE/Military — **та сама дизайн-система, але протилежний емоційний тон**. Військовий каталог — похмурий ч/б, тактичні силуети, боєць. Комерційний — яскравий колір, чистий клієнтський простір, «родина/бізнес». Стор. 2 військового відкривається гарантією; комерційного — «FAMILY SOLUTIONS FOR SUCCESS». Один продуктовий ряд → **два каталоги під дві аудиторії**.

**Що взяти для AMTarget:** це пряме підтвердження Варіанту Б зі структури — **два каталоги (Спорт/Цивільний + Military)** з різною обкладинкою, фото й мовою, але спільною версткою й брендом.

## 7. Action Target — Store Catalog 2025 (28 стор.)
**Архетип: легкий роздрібний каталог витратних товарів**

**Обкладинка:** мінімалізм — світло-сірий фон, великий чорний логотип-приціл AT по центру, вертикально «STORE».

**Особливе:** стор. 2 — **діаграма «ACTION TARGET ADVANTAGE»**: процес із піктограмами Consult → Design → Engineer → Manufacture → Install → Service → Supply → Train. Тобто навіть роздрібний каталог продає «повний цикл під ключ» як візуальний процес.

**Що взяти для AMTarget:** **піктограмна діаграма «повного циклу»** — ідеально лягає на USP AMTarget «виробник повного циклу, лазерне різання ±0.1 мм» (сталь → різання → зварювання → фарбування → доставка).

## 8. Action Target — AutoTargets Brochure (14 стор.)
**Архетип: глибока брошура одного продукту**

**Обкладинка:** чисте фото тиру з роботизованими рухомими мішенями, синя смуга з «AUTOTARGETS» внизу. 14 сторінок присвячено одному продуктовому ряду.

**Особливе:** навіть брошура одного продукту **всередині ділиться за аудиторією** — окремі розділи `Law Enforcement & Military` і `Commercial Ranges`, плюс `Advanced Controls`, фото продукту з планшетом-керуванням.

**Що взяти для AMTarget:** формат «глибокої брошури» для флагмана (напр. серія Combat або Texas Star) — коли одного SKU достатньо для окремого міні-документа під конкретного замовника.

## Висновки й рекомендації для каталогу AMTarget

**1. Відкривати довірою, а не товаром.** Усі найсильніші (AT, Meggitt) починають із гарантії/стандартів/цифр. Для AMTarget: перша змістовна сторінка — «10 000+ мішеней · виробник повного циклу · підрозділи ЗСУ», а не одразу прайс.

**2. Два тони під дві аудиторії.** Action Target прямо доводить: один продуктовий ряд — два каталоги (Спорт = яскравий колір; Military = стриманий, тактичний). Це підтверджує Варіант Б нашої структури.

**3. Гібридна сітка під обсяг лінійки.** У AMTarget 196 SKU. Рішення: флагмани (Combat, IPSC-гонг, Texas Star, Поппер) — за принципом Eemann «1 продукт = 1 сторінка» з блоком «калібр/дистанція»; масові варіації (круглі гонги 50–350, тарілки) — за принципом Birchwood Casey «сітка 3 колонки з артикулом».

**4. Військовий one-pager за шаблоном Meggitt.** Окремий вкладний лист під Prozorro/ЗСУ: Опис → Переваги → Застосування → Специфікації (сталь/HBW/калібр) → Контакт, з цифрами-доказами й згадкою стандартів (IPSC-відповідність).

**5. Діаграма «повного циклу» (з Action Target Store).** Піктограмний процес Сталь → Лазерне різання ±0.1 мм → Зварювання → Фарбування → Контроль → Доставка — візуалізує головний USP AMTarget.

**6. Відкрита таблиця цін за обсягом (з QT).** Для B2B/військових — рівні 1 / 5 / 10+ шт або «гурт за запитом». Знижує тертя великого замовлення.

**7. Технічні прийоми верстки, які варто запозичити:**
- блок «Протестовано: калібр / дистанція» на кожному флагмані (Eemann);
- кругла вирізка-callout для зони ураження (Eemann);
- вертикальні «вкладки» розділів на краю сторінки для навігації (Action Target);
- кольорові «прапорці» переваг на фото — «AR550», «Hardox-клас», «для ЗСУ» (Birchwood Casey);
- вигодо-орієнтовані заголовки замість сухих назв SKU (Action Target).
