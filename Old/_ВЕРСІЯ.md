# Old — оригінальна версія лендінгу (before)

Це **ізольований, самодостатній знімок** першої версії сайту — *до* редизайну другого блоку.
Зроблено для порівняння «before / after». Має власну копію `assets/` (шрифти, фото, CSS),
тож відкривається незалежно від робочої версії.

**Відкрити:** `Old/index.html` у браузері.
**Робоча (поточна) версія:** `../landing frame/index.html`.

## Чим відрізняється від поточної (що змінювали в другому блоці «SOCIAL PROOF»)
- надзаголовки мали **номери-бейджі** (01, 02…); тут вони на місці;
- **H2 більші** (до 44px), центровані;
- другий блок **вертикальний**, цифри в суцільній **плашці-рамці** з жовтим світінням;
- картки бригад — порожні **круглі** заглушки + підписи «Бригада 01–06», без лого та скосів.
